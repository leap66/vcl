认证模块

  • 欢迎界面
    1. 根据物理缓存判断是否进入引导页界面
    2. 根据Token物理缓存判断是否已登陆
  • 引导页界面
    1. 三张引导图，截取聊天、语音控制、设置
    2. 点击立即使用进入登录页
  • 登录界面
    1. 手机号为11位，密码6-16位
    2. 密码可切换是否可见，默认不可见
    3. 点击【登录】，网络请求成功后进入【主界面】
    4. 点击【忘记密码】，进入【修改密码界面】
    5. 点击【新用户注册】，进入【手机号效验界面】
    6. 点击【QQ登录】，关联QQ并登录
    7. 点击【微信登录】，关联微信并登录
  • 手机号效验界面
    1. 输入11为手机号后，激活获取验证码按钮
    2. 点击【获取验证码】，网络请求成功后，焦点切换到验证码输入框
    3. 点击【下一步】，网络请求成功后，进入【注册界面】
  • 注册界面
    1. 昵称输入1-12位
    2. 密码输入6-16位
    3. 点击【下一步】，网络请求成功后，调用退出事件
  • 修改密码界面
    1. 密码输入6-16位
    2. 验证码输入6位
    3. 点击【下一步】，网络请求成功后，调用退出事件
  • 修改手机号界面
    1. 新手机号输入11位后，激活获取验证码按钮
    2. 验证码输入6位
    3. 点击【下一步】，网络请求成功后，调用退出事件

聊天模块

  • 聊天界面
    1. 可下拉获取更对聊天记录
    2. 可发送语音、文字、图片信息
    3. 点击更多可查看详细信息
  • 小幕聊天界面
    1. 可下拉获取更对聊天记录
    2. 可发送语音、文字、图片信息
    3. 点击更多可查看详细信息
    4. 可智能获取笑话、新闻、故事等
  • 选择好友聊天界面
    1. 网络查询最近聊天记录并显示
    2. 可选择多选模式
    3. 点击【好友项】，二次确认后发送信息
    4. 点击【创建新的聊天】，跳转至【选择联系人界面】
  • 选择联系人界面
    1. 可多选模式
    2. 网络查询通讯录好友
    3. 点击【搜索】，可在当前界面网络请求搜索好友或群聊
    4. 点击【选择群聊】，跳转至【群聊界面】
    5. 点击【好友项】，即为选中该好友
    6. 点击【完成】，根据type二次确认后发送信息 或 跳转至【聊天界面】
  • 群聊界面
    1. 可多选模式
    2. 点击【群聊项】，即为选中该群聊
    3. 点击【完成】，根据type二次确认后发送信息 或 跳转至【聊天界面】

    3. 点击【雷达加好友】，跳转至【雷达加好友界面】
    4. 点击【面对面建群】，跳转至【面对面建群界面】
    5. 点击【扫一扫】，跳转至【扫一扫界面】
    6. 点击【手机联系人】，跳转至【手机联系人界面】
    7. 点击【公众号】，跳转至【搜一搜界面】
  • 雷达加好友界面
    1. 显示雷达搜索动画
    2. 搜索到信息后，跳转至【好友资料界面】
  • 面对面建群界面
    1. 和身边的好友输入同样的4个数字，可进入同一个群
  • 手机联系人界面
    1. 显示手机联系人
  • 好友资料界面
    1. 点击【头像】，可显示好友头像弹框
    2. 点击【发消息】，跳转至【聊天界面】
    3. 点击【更多】，跳转至【好友资料设置界面】
  • 好友资料设置界面
    1. 点击【设置备注和标签】，跳转至【设置备注和标签界面】
    2. 点击【推荐给好友】，跳转至【选择好友聊天界面】
    3. 可设置为星标朋友，静默网络请求
    4. 可设置为不让他看我的朋友圈，静默网络请求
    5. 可设置为不看他的朋友圈，静默网络请求
    6. 可加入黑名单，静默网络请求
    7. 可投诉，跳转至【投诉界面】
    8. 可删除该好友，二次确认后网络请求删除
  • 设置备注和标签界面
    1. 备注名1-12位
    2. 点击【标签】，跳转至【标签界面】
    3. 电话号码11位
    4. 描述0-100位
    5. 备注名1-12位
    6. 点击【添加图片】，跳转至【添加图片界面】
  • 投诉界面
    1. 选择投诉原因并二次确认后网络请求该投诉
  • 群聊资料界面
    1. 网络请求所有群成员信息并显示
    2. 可增加并邀请好友加入该群
    3. 点击【群聊名称】，跳转至【输入设置界面】
    4. 点击【我的本群昵称】，跳转至【输入设置界面】
    5. 点击【群二维码】，跳转至【二维码界面】
    6. 点击【群公告】，跳转至【群公告界面】
    7. 点击【投诉】，跳转至【投诉界面】
    8. 点击【清空聊天记录】，二次确认后可清空该群聊天记录
    9. 点击【删除并退出】，二次确认后可删除并退出该群
    10. 可设置消息免打扰、置顶聊天、保存到通讯录、显示群成员昵称操作
